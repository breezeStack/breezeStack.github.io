Eclipse报错：eclipse ClassNotFoundException:org.jbpm.api.ProcessEngine

Created2021-02-08|Updated2021-03-11|工作
|Post View:

一、问题描述

二、解决办法

1、查看本地是否maven仓库（找项目开发人员要来的）中是否存在这个包

2、发现存在，查看pom.xml文件中是否存在这个依赖，发现没找到jbpm-api，于是对照着本地仓库中的文件夹地址手动添加

3、添加后保存pom.xml
