解决jacob使用遇到问题_Dispatch not hooked to windows memory

Created2021-06-16|Updated2021-06-16
|Post View:

1

今天使用jacob读取word文档中的表格内容报异常：Dispatch not hooked to windows memory

仔细单步调试发现在for循环中第二次循环时之前的Dispatch变为空，需要重新赋值

在重新赋值之前先判断是否为空

在释放进程时也需要判断
