svn: E155004: There are unfinished work items in *** run 'svn cleanup' first.

Created2021-06-06|Updated2021-06-18|工作
|Post View:

1、遇到问题：

在eclipse上开发项目的时候，svn突然更新不了，报错

svn: E155004: There are unfinished work items in ‘D:\Tools\Eclipse\Eclipse_2020\Data\Feezet_Project\XAPF_XMPF\xmpf’; run ‘svn cleanup’ first.

2、解决问题：

百度了一下，发现是被锁住了，于是在eclipse里的Team中进行svn cleanup结果还是报同样的错误，于是到目录下进行cmd svn cleanup，发现电脑没安装svn，于是上官网安装

安装后在项目目录下右键打开svn的cleanup，选择Break write locks，clean完成提示success

再更新项目就可以了

参考网址：svn: E155004: There are unfinished work items 问题解决 https://blog.csdn.net/gjp014/article/details/84857016
